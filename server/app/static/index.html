RiskyBuyer

A shared blacklist of problematic customers for online stores.

What it is

RiskyBuyer is a community list of problematic customers (fake orders, uncollected cash-on-delivery parcels, abusive behaviour), keyed by phone number or name. Stores can check an incoming order against the shared list and flag risky buyers before shipping.

How to use it

It powers the free Problem Client WooCommerce plugin. In the plugin, open WooCommerce → Problem clients → Settings and enable synchronization. Reading the shared list is open; writing requires an authorized API key.

API base: https://riskybuyer.com/v1

Privacy & fairness

The list contains only contact identifiers and a short reason — no sensitive personal data. Synchronization in the plugin is opt-in. To request correction or removal of an entry, see the Privacy Policy.
