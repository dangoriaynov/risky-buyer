Privacy Policy

What data is stored

For each entry the Service stores: a phone number and/or name (as a contact identifier), a reason code (uncollected / fake / abusive / other), an optional short note, the domain of the submitting store, a status, and a timestamp. No payment data, addresses, or other sensitive personal data are stored.

How data is collected

Data is submitted by stores running the Problem Client plugin that have explicitly enabled synchronization and hold a write API key. Synchronization is opt-in and off by default. Reading the shared list is open to plugin users to check incoming orders.

Purpose and legal basis

Data is processed for the legitimate interest of preventing order fraud and losses for small online stores. Stores submitting data are responsible for having a lawful basis under their local law.

Retention

Entries are kept while relevant. Removed entries are flagged and propagated to subscribers so they stop matching.

Your rights / removal requests

To request access, correction, or removal of an entry about you, contact the operator at privacy@riskybuyer.com with the phone/name in question. Verified requests are actioned promptly.

Sharing

The shared list is available to authorized plugin installations for the sole purpose described above. It is not sold.
